## What I do

I specialize in **Ruby on Rails** with a strong focus on the frontend layer — **Hotwire (Turbo + Stimulus)**, ViewComponents, and modern CSS. I believe Rails is a full-stack framework, and I treat the UI with the same care as the backend.

When I'm not shipping production Rails apps, I build fun side projects with **React**, **React Native**, and **TypeScript** to stay sharp and explore new ideas.

- **Tech:** React Native, Expo, TypeScript, Skia
- **Highlights:** Procedurally generated tree visuals with animated animals, smart relationship engine, fully offline, JSON import/export

### grafik-sterylizacja
